# 울산애니원고등학교 C# 수업 강의

울산애니원고등학교에서 진행한 **C# 프로그래밍 입문 수업** 자료입니다.
기초 문법부터 객체지향, 제네릭, 델리게이트/LINQ, 스레드까지 6개 강의로 이어집니다.
각 강의는 독립적으로 실행되는 Visual Studio 콘솔 솔루션(`Lecture_N.sln`)이에요.

## 📚 커리큘럼

| 강의 | 주제 | 다루는 내용 |
|---|---|---|
| **Lecture 1** | 기초 문법 | 기본 자료형, 변수 선언, `enum` |
| **Lecture 2** | 클래스 & 객체 | 클래스, 생성자, 생성자 체이닝(`: this()`), 실습 문제(1~10) |
| **Lecture 3** | 상속 | 상속, 접근 제한자(`public`/`protected`/`private`), base ↔ derived |
| **Lecture 4** | 배열 & 제네릭 | 배열, 메서드 오버로딩, 제네릭 `<T>`, 제네릭 컬렉션 `MyList<T>` |
| **Lecture 5** | 델리게이트 & LINQ | 델리게이트, 프로퍼티, 람다 · 표현식 멤버(`=>`), LINQ |
| **Lecture 6** | 스레드 | `Thread`, `Sleep`, `Abort` — 동시성 기초 |

## ▶️ 실행 방법

각 강의 폴더의 `.sln` 파일을 **Visual Studio**로 열어 실행하면 됩니다.

```
Lecture_1/Lecture_1.sln
Lecture_2/...
...
Lecture_6/Lecture_6.sln
```

## 🛠 환경

- **언어** — C# (.NET 콘솔 애플리케이션)
- **도구** — Visual Studio 2017 이상
